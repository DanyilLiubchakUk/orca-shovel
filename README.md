# OrcaShovel

OrcaShovel is a TanStack Start app scaffolded with Bun, React, TypeScript, and Tailwind CSS.

This repository is intentionally small at the first commit. Product planning documents and AI architecture notes will be moved in after the clean application baseline is committed.

## Commands

Run the app locally:

```bash
bun --bun run dev
```

Build for production:

```bash
bun --bun run build
```

Run tests:

```bash
bun --bun run test
```
